Faith Integration

Islamic Integration.

Custom-built tools designed to keep you connected to your faith, seamlessly integrated into your daily workflow.

Four Custom Hidayah Apps

Hidayah Prayer Times

Auto-detects location via IP (falls back to Makkah). Calculates 5 daily prayers using the Muslim World League (MWL) method. Features desktop notifications, Azan audio alerts, and real-time Conky dashboard updates.

Hidayah Quran Reader

4 surahs (Al-Fatiha, Al-Ikhlas, Al-Falaq, An-Nas) available OFFLINE out of the box. Automatically downloads all 114 surahs on first internet connection. Features Uthmani script and Sahih International translation with local caching.

Hidayah Khal Wrapper

Terminal-based Hijri/Gregorian calendar system. Displays dual calendars with custom Hidayah branding (Gold/Green). Accessible directly from the application menu for non-technical users.

Conky Desktop Widget

An always-visible dashboard showing prayer times, Hijri date, system metrics, and 8 rotating inspirational Quran verses in elegant gold calligraphy style on a transparent background.

System-wide Features

Premium Arabic Fonts

Noto Kufi Arabic and Noto Sans Arabic pre-installed and configured for perfect legibility system-wide.

Islamic Bookmarks

Quick access to Quran.com, IslamQA, and Sunnah.com pre-loaded in Firefox ESR bookmarks toolbar.

Bismillah Login

Custom SDDM and LightDM greeting screens featuring 'Bismillah' in beautiful Arabic script.

Native ar_SA Locale

Active Protection

6-Layer Islamic Shield.

Hidayah OS implements a multi-layered security stack to ensure your browsing remains clean and private.

1

Layer 1 — CleanBrowsing Islamic DNS

ACTIVE

Blocks 10M+ haram domains at DNS level. Primary: 185.228.168.10 | Fallback: 185.228.169.10. DNS over TLS + DNSSEC enabled.

2

Layer 2 — nftables Islamic Blocklist

ACTIVE

578 specific haram/gambling/alcohol domains. Blocked at kernel level — below application layer.

3

Layer 3 — DNS Hijack Prevention

ACTIVE

Forces ALL DNS through CleanBrowsing. Even VPN cannot bypass this layer.

4

Layer 4 — Firefox AppArmor Profile

ACTIVE

39 deny rules — prevents browser tampering. Cannot disable security settings without Guardian PIN.

5

Layer 5 — Firejail Browser Sandbox

ACTIVE

Firefox runs in isolated container. WebRTC disabled — no IP leaks possible.

6

Layer 6 — SafeSearch Enforcement

ACTIVE

Google, Bing, YouTube, DuckDuckGo SafeSearch locked via Firefox policies.
